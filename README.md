# 🛡️ Aegis

> A privacy-first personal life system that protects, organizes, and assists — with user-controlled data and AI that works for you, not against you.

---

## 🧠 What is Aegis?

Aegis is a local-first, privacy-focused platform designed to help you manage every aspect of your digital life — securely, intelligently, and under your full control.

It combines:

- 🔐 Data protection  
- 🧠 AI assistance  
- 🧩 Modular services  
- 🛠️ Workflow automation  
- 👤 User-controlled privacy  

All in one system.

---

## 🔐 Core Philosophy

Aegis is built on a simple idea:

> You cannot trust a system unless it cannot betray you.

That means:

- Your data is yours — always  
- Sensitive data is protected with zero-knowledge encryption  
- No one (including admins) can access your private data  
- Nothing is shared unless you explicitly choose to share it  

---

## 🤖 AI That Works For You

Aegis includes an AI assistant designed to:

- help you organize and understand your data  
- assist with everyday tasks  
- provide insights and recommendations  

But:

- it only uses data you allow  
- it never shares your data externally without your approval  
- it respects all privacy boundaries  

---

## 🛡️ Privacy & Protection Features

- 🔐 Zero-knowledge encryption for sensitive data  
- 🛡️ Auto-Protect Mode (your data bodyguard)  
- ⚠️ Risk-aware third-party integrations  
- 🧠 Informed consent before data sharing  
- 🔍 Transparent security auditing  
- 📜 Public security reports  

---

## 🌐 Third-Party Integrations

Aegis allows integrations (email, cloud, etc.), but:

> External services operate outside Aegis.

Before any data is shared:

- You are shown exactly what is being shared  
- You are shown the risks  
- You must explicitly approve it  

No silent data flow. Ever.

---

## 🧩 System Architecture

- ⚙️ Raw PHP (no heavy frameworks)  
- 🧠 Front controller + modular system  
- 🔐 RBAC (role-based access control)  
- 🗄️ SQLite (default) or MySQL  
- 🔒 Vault system for encrypted data  
- 🔌 API-driven modules

## 🚀 Installation

bash git clone https://github.com/phreakin/aegis.git cd aegis 

Then run the installer:

bash php install.php 

Or open in browser:

text http://localhost/install 

Follow the guided setup:

- environment check  
- database setup  
- admin account  
- privacy configuration  
- AI policy agreement  

---

## ⚠️ Important

If you enable zero-knowledge encryption:

> If you lose your password or recovery key, your data cannot be recovered.

This is by design.

## 🧠 Why Aegis?

The name comes from ancient mythology:

- The Aegis was a shield carried by Zeus and Athena  
- It symbolized protection, wisdom, and strength  

That’s exactly what this system is built to provide.
